Welcome this GitHub Pages.

This web page is just a new hobby to try to analysis the data published by hkupop on the 2016 rolling survey of the public's view on the upcoming Legco. You can get those data in SPSS format or csvy here. In fact it is lonely if I am the only guy doing it.
